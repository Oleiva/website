The folder in which your layout documentation is supposed to be generated into. Path relative to your `package.json` file.

```json
{
  "name": "your site",
  "version": "1.0.0",
  "description": "",
  "cuttlebelle": {
    "folder": {
      "docs": "docs/"
    }
  }
}
```
