The name and location of the default partial layout. Path relative to your `code` folder.

The `.js` extension should be omitted.

```json
{
  "name": "your site",
  "version": "1.0.0",
  "description": "",
  "cuttlebelle": {
    "layouts": {
      "partial": "partial"
    }
  }
}
```
