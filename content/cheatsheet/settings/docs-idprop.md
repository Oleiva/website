To allow us to auto-build the docs we need to fill the default props with some examples.

This is the `_ID` sample data.

```json
{
  "name": "your site",
  "version": "1.0.0",
  "description": "",
  "cuttlebelle": {
    "docs": {
      "IDProp": "page2"
    }
  }
}
```
