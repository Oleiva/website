Cuttlebelles design is at heart of progressive enhancements. Even if you strip all the code away, you still have readable markdown files that can be used
elsewhere. Cuttlebelle makes websites out of them. Now the next step is to add a GUI on top of Cuttlebelle that will make it easy for you to add partials and
fill in its props.

The **proposal** so far:
